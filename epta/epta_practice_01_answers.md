## Task A — RT Phraseology (Q1 ~ Q3)

### Q1. Emergency Situation Handling
- **내 답변:**
Mayday, Mayday, Mayday. Tower, November 123AB. Engine number one have severe vibration after takeoff, we reduce power on that engine and climbing to safe altitude now. Intention is return to airport for immediate landing. Three souls on board, fuel two hours. Request vector to nearest runway and emergency vehicle standing by.

**⚠️ 확인 필요 표현:**
- "Tower, November 123AB" — MAYDAY×3 직후 호출대상(Station)과 콜사인을 붙이는 순서/포함 여부
- "Intention is return to airport for immediate landing" — 의도 전달 정형 문구의 정확한 형태
- "Three souls on board, fuel two hours" — "souls on board"는 일반적으로 쓰이는 표현이나, 연료 표현 순서·단위(예: "fuel remaining two hours" 등)
- "Request vector to nearest runway and emergency vehicle standing by" — 한 교신에 요청 두 개를 묶는 방식이 표준 권고 순서와 맞는지

일반적으로 이렇게 쓰이나, 한국교통안전공단 공식 샘플테스트 자료로 반드시 교차검증 권장.

### Q2. Weather Avoidance
- **내 답변:**
Approach, November 123AB, request weather deviation. We have big cumulonimbus about twenty mile ahead, tops is higher than our altitude, and we see turbulence and lightning, so we cannot continue this heading. Request deviate twenty degree left of track for avoid the weather, and if possible climb to flight level three five zero to stay above the cloud tops. Will advise when able resume own navigation back to original route.

**⚠️ 확인 필요 표현:**
- "request weather deviation" — 표준 권고 표현은 "request deviation due weather" 형태일 가능성
- "Request deviate twenty degree left of track" — "deviation"(명사)과 "deviate"(동사) 중 이 문맥에서 관용적으로 쓰이는 형태
- "Will advise when able resume own navigation" — "resume own navigation"은 실제로 쓰이는 표현이나, 벡터 종료 상황이 아닌 기상 회피 상황에서도 동일하게 쓰이는지

일반적으로 이렇게 쓰이나, 한국교통안전공단 공식 샘플테스트 자료로 반드시 교차검증 권장.

### Q3. System Failure
- **내 답변:**
During descent we saw hydraulic pressure indication go to zero and hydraulic low pressure warning light come on. First we run Hydraulic System Failure checklist from QRH, we select alternate hydraulic system if it is available, and we check flight control still respond normal. Then we declare urgency to ATC: "Pan-pan, pan-pan, pan-pan, [Station], November 123AB, hydraulic system failure, request priority landing and longer final, flight control and landing gear extension may be affect." We also call company operations on company frequency and advise about the failure, so maintenance and ground crew can be ready when we arrive. We brief cabin crew for possible manual gear extension and landing can be more firm than normal, and we prepare for go-around if gear is not confirm down and locked.

**⚠️ 확인 필요 표현:**
- **Pan-pan vs Mayday** — 완전한 유압 상실이 실제로 urgency(Pan-pan) 등급인지 distress(Mayday) 등급인지는 상황·기종·매뉴얼에 따라 다를 수 있음 (가장 우선적으로 교차검증 필요한 항목)
- "request priority landing and longer final" — 요청 두 개를 한 문장에 묶는 방식이 표준 순서와 일치하는지
- "flight control and landing gear extension may be affect" — 비정상 상황 통보 시 표준 권고 표현 형태

일반적으로 이렇게 쓰이나, 한국교통안전공단 공식 샘플테스트 자료로 반드시 교차검증 권장.

## Task B/C — Picture Description & Discussion (Q4 ~ Q7)

> 아래 답변에 대한 언어적 관찰: 짧고 단순한 문장 위주이며, 관사 누락("de-icing truck" → "the de-icing truck" 등)과 시제·단복수 불일치("person spray"), 콜로케이션 오류("more firm than normal")가 반복적으로 나타남. 다만 의미 전달과 논리 전개(상황 묘사 → 원인/영향 → 대책) 자체는 일관되게 유지되고 있음. Q4~Q5에 포함된 항공 용어(fluid type, holdover time, Local Standby/Full Emergency 등)는 phraseology가 아닌 일반 지식이지만, 최신 AIP·사내 매뉴얼과 대조해 정확한지 확인하는 것을 권장.

### Q4. Scenario Analysis A (De-icing)
- **내 답변:**
In this picture, I see airplane, it is big airplane, parking near the de-icing area in the airport. There is de-icing truck next to the airplane, and one person spray liquid on the wing, color is orange, I think this is de-icing fluid, maybe Type one, for remove ice and snow before takeoff, this concept we call clean aircraft concept, wing and other critical surface must be clean before takeoff. Weather look cold, maybe winter time, some snow is around.

About safety, if airplane not de-iced well, ice can stay on the wing and this is dangerous for takeoff because lift can be not enough and control surface can be affect too. Also if fluid put on too early and takeoff is delay, fluid protection time can finish, this is call holdover time. For prevent this, ground crew must follow holdover time table for the weather condition, and tell the pilot exactly when de-icing finish and what fluid type they use. Pilot also can request wing check or pre-takeoff contamination check one more time before takeoff if not sure.

### Q5. Scenario Analysis B (Airport Emergency Response)
- **내 답변:**

### Q6. Human Factors & Automation
- **내 답변:**
I think automation is very helpful for pilot, because pilot don't need to do everything by hand, especially in long flight, automation can help for communication, navigation, and monitor the system same time. But if pilot depend on automation too much, this is not good, because pilot manual flying skill can go down if they don't practice, and also pilot can have automation complacency, this mean pilot trust automation too much and don't notice fast if automation make mistake, this is call automation bias. For example if autopilot doing something wrong and pilot trust one hundred percent, pilot cannot react on time, and this can be dangerous especially in critical phase like approach and landing. I think pilot should use automation for normal situation, but also must practice manual flying sometimes, like in simulator or when workload is low, for keep the skill and situation awareness. Airline also should make training have more manual flying and hand-flying practice, not only automation scenario.

### Q7. Lithium Battery Regulations
- **내 답변:**
I think lithium battery is very dangerous for airplane because it can fire very fast, and this fire, they call thermal runaway, is hard to stop with normal fire equipment, especially in cargo compartment because access is difficult during flight. I think airline and ICAO, FAA must make rule more strict, this rule already exist in ICAO Technical Instructions and IATA Dangerous Goods Regulation, but I think it must be follow more strict, for example battery state of charge must be limited when it carry as cargo, container must be fire resistant or fire containment bag must be use, and cargo worker and cabin crew must have training for know what to do if battery become hot or smoke happen. Also I think cargo compartment need better sensor, like heat or smoke sensor, so crew can know early if problem happen. Airline also must tell passenger clearly about rule, spare battery must be carry in cabin, not in check baggage, because in cabin crew can respond fast if problem happen.
